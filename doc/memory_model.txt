Программа на BitVaram включает в себя следующие уровни памяти:

## Регистровая память.

Представляет собой 3 регистра для буферизации стандартных потоков, счетчик и регистр флагов.

---

#Регистр стандартного вывода:
По умолчанию имеет размер 1 байт, использует ascii-подобную кодовую таблицу. Для вывода используются зарезервированные текстовые команды. Регистр имеет два состояния: ожидание и готовность. В режиме готовности регистр содержит в себе ненулевой код, доступный для вывода. В режиме ожидания регистр содержит специальное зарезервированное значение - нулевой байт, в таком случае вывод прекращается.

#Регистр стандартного ввода:
По умолчанию имеет размер 1 байт, использует ascii-подобную кодовую таблицу. Для ввода используются зарезервированные текстовые команды. Регистр имеет два состояния: ожидание и готовность. В режиме готовности регистр содержит в себе нулевой код, недопустимый для вывода. В режиме ожидания регистр содержит в себе нулевой код, недопустимый для вывода. В режиме готовности регистр содержит в себе ненулевой код, символ немедленно выводится - регистр переходит в состояние ожидания.

#Счетчик:
по умолчанию имеет размер 2 байта, содержит в себе код неотрицательного числа. Для изменения используются зарезервированные текстовые команды (for-цикл, подробнее в syntax.txt). Используется для организации циклов.

#Регистр флагов:
По стандарту языка имеет размер 16 бит. Для установки флагов используются зарезервированные текстовые команды (не все флаги доступны для ручного изменения). Может использоваться для организации ветвления (if-условный оператор, подробнее в syntax.txt), и регистрации ошибок.

---

## Лента

---

Переменные, с которыми не происходит вычислений, хранятся на гипотетически бесконечной ленте - байтовом векторе. В runtime память для переменной выделяется в бесконечном цикле (как operator_new() в C++), номера использованных ячеек и смещения хранятся в специальной структуре.
По умолчанию длина ленты составляет 1024 байта.

---

## Стек

---

Переменные в момент вычислений хранятся в специальной стек-ориентированной структуре. По умолчанию стек имеет размер 64 байт. Переменные, объявленные в теле функции имеют ограниченное время жизни, эти переменные - единственная категория, за которой ведется автоматический контроль, для остальных необходимо выполнить разопределение.
